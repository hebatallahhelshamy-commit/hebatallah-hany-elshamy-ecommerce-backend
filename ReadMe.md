# E-Commerce API Project

مشروع باك إند بسيط معمول بـ Node.js و Express و MongoDBعشان يدير عمليات موقع تسوق أونلاين.

---

## 1. عن المشروع ومميزاته (Features)
المشروع ده بيشغل سيرفر لموقع تجارة إلكترونية، وفيه الحاجات دي:
1 الأقسام (Categories):** بنضيف ونمسح ونعدل الأقسام.
2 المنتجات (Products):** بنعرض المنتجات وبنربطها بالأقسام بتاعتها.
3 السلة (Cart):** بتضيف المنتجات، وبتحسب السعر الإجمالي لوحدها تلقائيًا (الكمية × السعر).
4 الطلبات (Orders & Checkout):** بتاخد الحاجات اللي في السلة وتعمل بيها فاتورة رسمية حالتها `Pending` وتفضّي السلة.
نظام الأخطاء: لو حصل أي خطأ في السيرفر، السيستم بيتعامل معاه من غير ما يقع.

---

## 2. إزاي تشغل المشروع عندك (Installation)

### الحاجات اللي محتاجها:
* يكون عندك Node.js على الكمبيوتر.
* يكون عندك حساب على MongoDB Atlas أو برنامج MongoDB Compass

### خطوات التشغيل:
1. نزل الفولدر عندك وافتحه في VS Code.
2. افتح الـ Terminal واكتب الأمر ده عشان تنزل كل المكتبات:
   ```bash
   npm install

## Files 

config/db.js - ملف الاتصال بالداتابيز والأخطاء.

controllers/ - جواه التحكم والمنطق بتاع كل جزء (المنتجات، السلة، ).

models/ - جواه الجداول وشكل البيانات في الداتابيز.

routes/ - جواه المسارات والروابط بتاعة السيرفر.

app.js - الملف الرئيسي اللي بيربط كل حاجة ببعضها.

server.js - الملف اللي بيشغل السيرفر.


## روابط الـ API (Endpoints)
الأقسام (Categories)
GET /categories - عشان تشوف كل الأقسام.

POST /categories - عشان تضيف قسم جديد.

المنتجات (Products)
GET /products - عشان تشوف كل المنتجات.

POST /products - عشان تضيف منتج جديد.

السلة (Cart)
POST /cart/add - عشان تضيف منتج للسلة وتزود الكمية ويحسب الإجمالي.

الطلبات والفاتورة (Orders)
POST /orders/checkout - عشان تشتري وتعمل الفاتورة وتفضّي السلة.
